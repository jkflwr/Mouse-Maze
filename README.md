Mouse Maze is a 2D maze game built using Java Swing. The player controls a mouse that navigates through maze levels to reach the cheese while avoiding walls.

How to Run:
Make sure you have Java installed (JDK 8 or higher)
Download or clone this repository

Compile the files:
javac *.java

Run the program:
java Project2Runner

How to Play:
Move your mouse cursor to choose the direction
Press:
W key OR
↑ (up arrow key)
The mouse character will move toward your cursor
Avoid walls and reach the cheese to win
